yi.Observable

描述 可观察的数据项

yi.Observable.setValue/yi.Observable.getValue

  • 说明

    Description
    设置，或获取数据项的值
    Parameters
    NameTypeDescription
    value * 要设置的值
    reason string 传递给事件用的，表明改变的原因，默认是value.set
    RETURNS*该数据项中装载的值

yi.Observable.object

  • 说明

    Description
    获取或指定该数据项在那个对象上。变更对象会以object.change为reason触发Observable的valuechange事件
    Parameters
    NameTypeDescription
    value object/function 对象
    source string 事件源
    RETURNSyi.Observable / object / function如果是设置，返回this;否者返回对象
